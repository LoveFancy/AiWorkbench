## Phase 5 详细规划

### 数据统计面板

- 按时间维度的趋势图（事件量、错误率、活跃用户）
- 按模型维度的使用统计（调用量、成功率、平均响应时长）
- 按用户维度的活跃度排行

### 告警规则

- 可配置的告警规则（如错误率 > 阈值、连续 N 分钟无事件上报）
- 告警通知渠道（企业微信 / 邮件）
- 告警静默期配置

### 使用报告导出

- 日报 / 周报 / 月报自动生成
- 导出格式：Excel / PDF
- 定时邮件发送

### 监控告警

- 接入 Prometheus + Grafana
- 系统级指标：CPU、内存、QPS、响应延迟
- 业务级指标：事件量、错误率、升级成功率
