# 自定义渠道模型多模态能力标识 — 后端接口需求

编辑时间：2026年6月11日

---

## 一、背景

客户端在"设置 → 模型配置（自定义）"中，用户添加华泰渠道后可点击"拉取模型"从供应商后端获取可用模型列表。拉取到的模型需要在 UI 中展示其多模态能力（"纯文本" / "多模态"标签），以便用户和 Agent 模式正确区分模型能力。

目前华泰大模型平台（`168.63.65.40:8090`）的 `/llm-service/v1/models` 接口**不返回**多模态能力标识字段，导致客户端无法自动区分。

## 二、需求

`GET /llm-service/v1/models` 接口在返回的每个模型对象中，新增一个标识该模型是否支持多模态的布尔字段。

**示例**（当前）：

```json
{
  "data": [
    { "id": "saas-doubao-15-pro-32k", "display_name": "saas-doubao-15-pro-32k" },
    { "id": "saas-doubao-seed-20-pro", "display_name": "saas-doubao-seed-20-pro" }
  ]
}
```

**期望**（新增字段，字段名任意，客户端侧做映射）：

```json
{
  "data": [
    { "id": "saas-doubao-15-pro-32k", "display_name": "saas-doubao-15-pro-32k", "supportsMultimodal": false },
    { "id": "saas-doubao-seed-20-pro", "display_name": "saas-doubao-seed-20-pro", "supportsMultimodal": true }
  ]
}
```
