# 客户端版本管理

## 版本号定义位置

客户端的版本号定义在 [apps/electron/package.json](../../apps/electron/package.json) 的 `version` 字段中：

```json
{
  "name": "@proma/electron",
  "version": "0.12.13",
  ...
}
```

## 运行时获取方式

通过 Electron 的 `app.getVersion()` API 获取，该 API 会自动读取 `package.json` 中的 `version` 字段。

## 版本号的初始化与使用

### 初始化入口

在 [apps/electron/src/main/index.ts](../../apps/electron/src/main/index.ts) 的 `bootstrap()` 函数中，启动时调用 `setPromaVersion(app.getVersion())` 将版本号注入到全局：

```ts
// apps/electron/src/main/index.ts:497
setPromaVersion(app.getVersion())
```

### 版本号存储与分发

`setPromaVersion` 定义在 [packages/core/src/providers/user-agent.ts](../../packages/core/src/providers/user-agent.ts)，将版本号存入模块级变量 `_promaVersion`，供全局使用：

```ts
let _promaVersion = '0.0.0'

export function setPromaVersion(version: string): void {
  _promaVersion = version
}

export function getPromaVersion(): string {
  return _promaVersion
}
```

### 版本号的使用场景

| 使用位置 | 用途 |
|---------|------|
| [packages/core/src/providers/user-agent.ts](../../packages/core/src/providers/user-agent.ts) | 生成 `Proma/x.x.x` 格式的 User-Agent 标识 |
| [apps/electron/src/main/lib/updater/auto-updater.ts](../../apps/electron/src/main/lib/updater/auto-updater.ts) | 更新检测时上报当前版本、版本方向校验 |
| [apps/electron/src/main/lib/observability-service.ts](../../apps/electron/src/main/lib/observability-service.ts) | 可观测性上报中携带客户端版本信息 |

## 版本号变更流程

提交代码时始终递增 `apps/electron/package.json` 中 `version` 字段的 patch 版本（如 `0.12.13` → `0.12.14`）。如果改动影响多个包，各包都要递增。
