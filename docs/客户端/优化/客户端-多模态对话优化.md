## 一、需求概述

后端返回的模型列表接口中，`ModelInfo` record 新增 `supportVision` 字段：

```java
public record ModelInfo(String id, String name, String description, String provider,
                         String baseUrl, Integer maxTokens, boolean enabled,
                         boolean supportVision) { }
```

- `supportVision: true` → 多模态模型
- `supportVision: false` → 纯文本模型

当前自定义模型配置（ChannelForm.tsx）中已用 `ModelCapabilityToggle` 组件展示模型的"纯文本"/"多模态"属性。本次改造需要让**泰为平台模型**也支持相同的展示。

## 三、改造方案

### 3.1 改动点一览

| # | 文件 | 改动类型 | 说明 |
|---|------|----------|------|

### 3.2 改动一：`model-service.ts` — 字段映射

**文件**：`apps/electron/src/models/model-service.ts`

**问题**：后端返回字段名为 `supportVision`，客户端使用 `supportsMultimodal`。由于 `httpGet` 返回的是 JSON 反序列化后的对象，字段名不匹配会导致 `supportsMultimodal` 始终为 `undefined`。

**方案**：在 `fetchUserModels()` 中解析响应时，将 `supportVision` 映射为 `supportsMultimodal`。有两种方式：

**方式 A（推荐）**：在类型层面兼容，定义解析函数做字段映射

```typescript
// 后端原始返回的模型结构
interface RawModelInfo {
  id: string
  name: string
  description?: string
  provider?: string
  baseUrl?: string
  maxTokens?: number
  enabled: boolean
  supportVision: boolean  // 后端字段名
}

// 在 fetchUserModels 中做映射
function mapRawModel(raw: RawModelInfo): ModelInfo {
  return {
    id: raw.id,
    name: raw.name,
    description: raw.description,
    provider: raw.provider,
    baseUrl: raw.baseUrl,
    maxTokens: raw.maxTokens,
    enabled: raw.enabled,
    supportsMultimodal: raw.supportVision,  // 字段映射
  }
}
```

**方式 B**：直接修改 `ModelInfo` 类型增加 `supportVision` 别名

```typescript
export interface ModelInfo {
  // ...
  /** 后端字段 supportVision，映射为 supportsMultimodal */
  supportsMultimodal?: boolean
  supportVision?: boolean  // 同时兼容两个字段名
}
```

然后在取值时做 fallback：`model.supportsMultimodal ?? model.supportVision`

> **建议采用方式 A**，保持类型纯净且映射逻辑清晰。

**具体改动**：

```diff
// apps/electron/src/models/model-service.ts

+/** 后端原始模型结构（字段名与客户端不同） */
+interface RawModelInfo {
+  id: string
+  name: string
+  description?: string
+  provider?: string
+  baseUrl?: string
+  maxTokens?: number
+  enabled: boolean
+  supportVision: boolean
+}

+function normalizeModels(rawModels: RawModelInfo[]): ModelInfo[] {
+  return rawModels.map((m) => ({
+    id: m.id,
+    name: m.name,
+    description: m.description,
+    provider: m.provider,
+    baseUrl: m.baseUrl,
+    maxTokens: m.maxTokens,
+    enabled: m.enabled,
+    supportsMultimodal: m.supportVision,
+  }))
+}

 // 在 fetchUserModels() 的成功分支中：
-  const { apiKey, models, total } = data.data
+  const { apiKey, total } = data.data
+  const models = normalizeModels(data.data.models as RawModelInfo[])
```

### 3.3 改动二：`PlatformModelsSection.tsx` — 平台模型增加多模态标签

**文件**：`apps/electron/src/platform-models/PlatformModelsSection.tsx`

**目标**：在 `PlatformModelRow` 中显示多模态标签，效果类似自定义模型但为**只读展示**（平台模型的多模态能力由后端决定，用户不可切换）。

**方案**：复用 `ModelCapabilityToggle` 的视觉样式，但改为只读显示。有两种方式：

**方式 A（推荐）**：抽取纯展示版本 `ModelCapabilityLabel`

在 `ChannelForm.tsx` 中新增一个只读展示组件：

```tsx
// 只读标签，仅展示模型能力，不可交互
export function ModelCapabilityLabel({ supportsMultimodal }: { supportsMultimodal?: boolean }): React.ReactElement {
  if (supportsMultimodal === undefined) return null
  return (
    <span className={cn(
      'inline-flex h-6 shrink-0 items-center rounded-md px-2 text-xs font-medium',
      supportsMultimodal
        ? 'bg-emerald-500/15 text-emerald-700 dark:text-emerald-300'
        : 'bg-muted text-muted-foreground'
    )}>
      {supportsMultimodal ? '多模态' : '纯文本'}
    </span>
  )
}
```

然后在 `PlatformModelRow` 中使用：

```tsx
// src/platform-models/PlatformModelsSection.tsx

import { ModelCapabilityLabel } from '@/renderer/components/settings/ChannelForm'

function PlatformModelRow({ model, onToggle }: ...): React.ReactElement {
  // ... 现有逻辑 ...

  return (
    <SettingsRow
      label={model.name}
      description={description || undefined}
    >
      <ModelCapabilityLabel supportsMultimodal={model.supportsMultimodal} />
      <Switch checked={model.enabled} onCheckedChange={onToggle} />
    </SettingsRow>
  )
}
```

**方式 B**：组件内联，不抽取

在 `PlatformModelRow` 中直接写一个简单的标签显示：

```tsx
{model.supportsMultimodal !== undefined && (
  <span className={cn(
    'inline-flex h-6 shrink-0 items-center rounded-md px-2 text-xs font-medium mr-2',
    model.supportsMultimodal
      ? 'bg-emerald-500/15 text-emerald-700 dark:text-emerald-300'
      : 'bg-muted text-muted-foreground'
  )}>
    {model.supportsMultimodal ? '多模态' : '纯文本'}
  </span>
)}
```

> **建议采用方式 A**，统一自定义模型和平台模型的标签样式，避免重复代码。
