# 工作区文件浏览器 — 人工测试用例

> 适用范围：右侧面板「工作区文件」与「会话文件」区域的文件树（`FileBrowser` 组件）。
> 执行方式：按用例逐条手工操作，记录「实际结果」并勾选「是否通过」。
> 测试环境：☐ macOS  ☐ Windows  ☐ Linux ｜ 版本号：________ ｜ 执行人：________ ｜ 日期：________

## 名词约定

- **文件树**：右侧面板中显示工作区/会话目录内容的区域。
- **根目录**：文件树的最顶层目录（工作区文件根 / 会话文件根）。
- **空白区域**：文件条目列表下方没有文件行的区域。
- **选中态**：行高亮（左侧出现主色竖条 + 浅色背景）。
- **Mod 键**：macOS 为 `Cmd`，Windows/Linux 为 `Ctrl`。
