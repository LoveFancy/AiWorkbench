代码审查助手

自动审查代码质量，发现潜在的安全漏洞、代码坏味道和性能问题。

功能

  • 安全审查 — 检测 SQL 注入、XSS、敏感信息泄露等安全问题
  • 代码规范 — 检查命名规范、代码结构、注释完整性
  • 性能分析 — 识别 N+1 查询、内存泄漏、不必要的重复计算
  • 最佳实践 — 推荐设计模式、错误处理、测试覆盖率建议

使用方式

在 Agent 对话中提及 @code-review 即可触发审查，或者直接说"帮我审查一下这段代码"。

环境变量

# 可选：指定审查规则集
REVIEW_RULESET=strict

# 可选：忽略特定规则
REVIEW_IGNORE=no-unused-vars,no-console

版本历史

  • v1.2.0 — 新增 Go 语言支持，优化 Rust 审查规则
  • v1.1.0 — 支持自定义规则集
  • v1.0.0 — 初始版本，支持 JS/TS/Python
