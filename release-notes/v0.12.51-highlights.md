# WorkMate v0.12.51 版本亮点

> 2026年6月 · v0.11.1 → v0.12.51

***

## 新特性

- **EIP 登录 & 默认模型自动分配** — 支持 EIP 账号登录，登录后自动分配默认模型，开箱即用
- **连接器生态（邮箱 / CLI 等）** — 支持华泰邮箱、CLI 等多种连接器，一键绑定即可在对话中直接查邮件、操作业务系统；Token 安全加密存储，启动即用
- **专家团系统** — 召唤多位专家协作完成任务，支持按分类筛选、重新召唤、下载&安装进度实时显示，大专家也不怕等
- **泰为SkillHub 技能市场** — 内置技能商店，可直接浏览、安装、删除技能，支持图片输入，按来源分类筛选
- **定时任务系统** — 设置周期性自动任务，完成后可推送到飞书通知，支持查看运行历史
- **HTML 静态预览** — 直接在 WorkMate 中预览 HTML 文件，支持手动刷新
- **工作区文件操作增强** — 支持分栏展示、Shift 连续多选、批量删除、文件跨会话移动、拖拽上传、键盘快捷键支持等
- **对话重试 & 历史回看** — 对话可重新生成，上下方向键快速调出历史消息
- **Auto 模式** — Agent 根据任务自动选择合适的工具和模型，无需手动切换，编辑保存更稳定

## 体验优化

- **启动更快** — 优化了启动流程，首屏加载速度明显提升，减少等待
- **操作更流畅** — 安装技能/专家不再卡界面，后台静默处理，装完即用
- **日志更省心** — 支持快捷的日志查看、问题上报，日志完全脱敏更安心
- **Git 操作优化** — 文件对比、提交等操作响应更快，中文路径正常显示
- **界面交互改进** — 侧边栏重新设计，Markdown 支持目录导航，首屏提前展示，滚动条更美观
- **Windows 兼容提升** — 修复中文路径乱码、文件改动误报等问题，WSL 体验更稳定
