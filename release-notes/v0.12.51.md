## v0.12.50 ~ v0.12.51（6月23日—6月24日，最新）

**新功能——HiAgent CLI 连接器**
- HiAgent CLI 连接器完整方案上线，支持默认连接器自动初始化
- 环境变量注入：Agent 启动/模型切换时自动注入连接器 env（HTSKILL_TOKEN / AGENTOS_ENV）
- Secret 管理：通过 Keychain/DPAPI 安全存储 Token，禁止明文落盘
- MCP 服务注册：token-check / hiagent-chat 等端点动态注册到 MCP 构建器
- 连接器 UI：Token/环境输入 → 步骤展示 → 重绑定
- CLI 连接器通过 enabled 状态判断 isConfigured（不再硬编码）
- 新增 DefaultConnectorInitStepId（check-runtime / install-skill）等类型扩展

**新功能——EIP MCP 服务**
- 新增 EIP 请求集成的 MCP Server 层（eip-request-mcp）与服务层（eip-request-service）
- hteip-client 新增 query 参数支持 POST + query 的 EIP 门户接口调用模式
- 同步更新 superpowers 插件至 5.1.2，新增 eip-skill-coach 技能
- 工作区同步增强：copyMissingConnectorFiles 同步增量文件，mergePresetConnectorEntries 保留 skillDirs

**新功能——DPAPI 预构建 & 启动性能优化**
- DPAPI 预构建从 node_modules 迁移至 resources/，增加多路径候选解析
- Bridges 延后至首屏加载完成后启动，减少启动竞争
- IPC 处理程序、Agent 事件总线、UI 组件均增加耗时追踪（性能诊断）
- 连接器管理：syncDefaultConnectorsToAllWorkspaces 始终同步并刷新预设元数据，保留用户启用状态
- MCP 构建器：空 selectedMcpServers 视为"全部加载"而非"一个都不加载"

**新功能——工作区增强**
- 工作区文件的复制、粘贴支持多选
- 支持 Shift 连续多选
- 支持工作区的批量删除
- 手册支持关闭

**优化**
- 优化首屏，提前首屏渲染
- 重构连接器页面，整体逻辑一致，都从本地文件读取
- 优化工作区代码结构
- 优化删除会话的提示词
- HiAgent 连接器重命名为"泰为智能体"
- 连接器选择器使用纯图标按钮、用户菜单卡片式重新设计
- 专家召唤按钮弹出面板
- 华泰邮箱连接器步骤/描述改进，支持 Enter 键提交
- 飞书关闭后，启动不再尝试重连

**Bug 修复**
- 解决工作区无法复制外部文件的 Bug（当复制了工作区内部的文件时）
- 修复 CURD 连接器与之前 MCP 逻辑冲突的问题
- 修复 Card 中导致 CLI 出现飞书解绑按钮的 Bug
- 修复复制粘贴的 Bug
- 加强了旧 MCP 文件的兼容逻辑
- 解决无法会话的问题
- 优化插件市场认证入口

**文档**
- 会话卡顿问题分析文档
- 新增泰为 HiAgent CLI 连接器设计方案文档
- 整理 Git 问题记录
