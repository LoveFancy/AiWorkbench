## 本月核心成果总结

| 分类 | 关键成果 |
|:----|:---------|
| **专家系统** | 从 WorkBuddy 专家团迁移 → 分类筛选 → 召唤重试 → 超大专家进度显示，全链路打通 |
| **SkillHub** | 市场上线 → Mock 支持 → 安装/删除 → 多模态输入 → 源分类，生态初步成型 |
| **自动化** | 定时任务调度系统（测试版）+ 飞书通知 + 运行历史跳转，实现闭环 |
| **日志体系** | 异步化改造 + 条件上报（不登录不上报）+ 启动时间上报 + 大幅减少冗余日志 |
| **启动性能** | 启动耗时优化方案完善 + 消息缓存消除读盘空窗 + Git diff 异步化 |
| **模型渠道** | 新增小米 MiMo、智谱 Coding Plan、MiniMax-M3 三个渠道，支持模型自动切换 |
| **UI/UX** | 工作区分栏、侧边栏大重构、Markdown TOC、HTML 预览、文件拖拽统一 |
| **Windows 兼容** | 中文路径乱码修复、CRLF 误报修复、WSL 检测增强、长路径统计修复 |
